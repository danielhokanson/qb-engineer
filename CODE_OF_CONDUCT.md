# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, as its
code of conduct for all participants in any project space — issues, pull
requests, discussions, code review, and any other channel maintained by
the project.

The full text of the Contributor Covenant 2.1 is available at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project, you agree to uphold the standards
described there.

## Reporting

If you experience or witness behavior that violates the Contributor
Covenant in any project space, report it to the project maintainer:

- Email: **daniel.hokanson@gmail.com**

All reports are reviewed promptly and kept confidential. The maintainer
will follow the enforcement guidelines in section 4 of the Contributor
Covenant.

## Scope

This code of conduct applies in all project-managed spaces (this repo
and all sibling repos: `qb-engineer`, `qb-engineer-ui`,
`qb-engineer-server`, `qb-engineer-deploy`, `qb-engineer-test`) and
when an individual is officially representing the project in public
spaces.
